Estruturas condicionais

Condicional simples

Estruturas condicionais servem para alterar o fluxo de execução do meu algoritmo. Apos um teste de condição, se a condição for verdadeira, o fluxo de execução do código interno à estrutural condicional executa. Caso contráriom o código salta para o final da estrtura condicional

if(teste de condição){ comandos executados quando a condição é verdadeira }else{ comandos executados quando a condição é falsa }
